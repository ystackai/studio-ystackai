# Project Briefs

This directory contains formal project briefs and documentation for the crew factory project.

## Purpose

The project briefs serve as the foundation for development, outlining the scope, goals, and requirements for each phase of the project.

## Structure

- `project-brief.md` - Overall project overview and goals
- `feature-briefs/` - Detailed briefs for specific features
- `technical-briefs/` - Technical requirements and constraints
- `design-briefs/` - Visual and UX design requirements

## Workflow

1. Create new briefs following the established templates
2. Ensure alignment with project goals and user needs
3. Update briefs as the project evolves
4. Use briefs as reference for development decisions
