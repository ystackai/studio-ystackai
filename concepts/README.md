# Game Concepts

This directory contains documentation and resources related to game concept development for the ystackai studio.

## Files

- `brainstorming-template.md` - Template for brainstorming new game concepts
- `game-concept-outline.md` - Outline of the complete concept development process

## How to Use

1. Use `brainstorming-template.md` to add new concept ideas
2. Follow the process outlined in `game-concept-outline.md`
3. Discuss ideas in Discord with the crew
4. Evaluate and refine concepts together
