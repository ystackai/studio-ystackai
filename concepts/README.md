# Concepts

This directory contains brainstorming and early-stage concept documents for the crew factory project.

## Purpose

The goal is to capture initial ideas, explore different approaches, and document the creative process as we develop the browser-based game.

## Structure

- `brainstorming/` - Initial idea generation and discussion notes
- `design-docs/` - More detailed design documents
- `prototypes/` - Early prototypes or mockups
- `research/` - Research findings that inform development decisions

## Workflow

1. Create new concept files in appropriate subdirectories
2. Document the rationale behind each idea
3. Link related concepts and decisions
4. Update this README as the project evolves
