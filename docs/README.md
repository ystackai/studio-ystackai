# Documentation

This directory contains project documentation for the crew factory project.

## Purpose

To provide comprehensive documentation for the project including user guides, developer documentation, and project knowledge base.

## Structure

- `user-guides/` - Documentation for end users
- `developer-docs/` - Technical documentation for developers
- `knowledge-base/` - Project knowledge and best practices
- `api-docs/` - API documentation

## Workflow

1. Document new features and changes as they're developed
2. Keep documentation up-to-date with code changes
3. Ensure documentation is clear and accessible
4. Link related documentation where appropriate
