# Art Style Guide

This document defines the visual direction and aesthetic principles for the game project.

## Core Aesthetic

The game will feature a **cyberpunk-inspired** visual style with:
- Neon color palette
- Dark, moody environments
- Futuristic, high-tech elements
- Organic textures mixed with digital interfaces
- Glitch effects and particle systems

## Color Palette

| Name       | Hex Code   | Usage                             |
|------------|------------|-----------------------------------|
| Neon Blue  | `#00F3FF`  | Primary highlights, UI elements |
| Electric Purple | `#8A2BE2` | Secondary accents, cyber elements |
| Deep Black | `#0A0A0A`  | Backgrounds, shadows            |
| Neon Pink  | `#FF00FF`  | Emphasis, alerts, UI focus        |
| Cyber Gray | `#4A4A4A`  | Mid-tones, UI backgrounds         |

## Typography

### Primary Font
- **Font Family**: 'Orbitron', sans-serif
- **Usage**: Headlines, UI text

### Secondary Font
- **Font Family**: 'Roboto', sans-serif
- **Usage**: Body text, descriptions

## Visual Elements

### Characters
- Futuristic humanoid designs
- Organic textures on armor and clothing
- Glowing details and neon accents
- Dynamic poses and expressions

### Environments
- Neon-lit cityscapes
- Rain-soaked streets
- High-tech buildings with holographic displays
- Glitch effects on digital surfaces

### UI Components
- Glass-morphism effects
- Neon borders and highlights
- Responsive elements with subtle animations
- Clear visual hierarchy

## Art Direction References

### Current References
- Cyberpunk 2077 art style
- Blade Runner 2049 visual motifs
- Neon Genesis Evangelion character designs
- Modern UI design trends (Figma, Material Design)

### Future Considerations
- Integration with game mechanics
- Consistency across all assets
- Adaptation for different screen sizes

## Implementation Notes

### Consistency
All visual assets should maintain consistent proportions, color usage, and stylistic elements to create a cohesive experience.

### Iteration
Visual assets will be iterated upon based on playtesting feedback and technical constraints.
