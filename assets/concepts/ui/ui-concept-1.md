# UI Concept: Main Menu

## Overview

The main menu UI for the cyberpunk-themed game. This interface follows the neon aesthetic with glass-morphism effects and dynamic neon highlights.

## Design Elements

- **Layout Structure**:
  - Centralized main menu with radial layout
  - Glass-morphism panels with subtle blur effects
  - Neon borders and glowing accents
  - Responsive elements that adapt to screen size

- **Visual Style**:
  - Primary: Deep black (`#0A0A0A`) backgrounds
  - Secondary: Neon blue (`#00F3FF`) for primary buttons and highlights
  - Accent: Electric purple (`#8A2BE2`) for secondary elements
  - Interactive: Neon pink (`#FF00FF`) for hover and active states

- **Typography**:
  - Primary font: 'Orbitron', sans-serif for headings
  - Secondary font: 'Roboto', sans-serif for body text
  - Text effects: Glowing text with drop shadows
  - Dynamic animations: Subtle text flickering effects

- **Interactive Elements**:
  - Buttons with glow effects on hover
  - Progress indicators with neon accents
  - Dropdown menus with animated transitions
  - Visual feedback for selections
