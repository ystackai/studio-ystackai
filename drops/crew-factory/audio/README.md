# Audio Assets for Crew Factory

This directory contains all audio assets for the Crew Factory drop.

## File Structure
- `sfx/` - Sound effects
- `music/` - Background music
- `vo/` - Voiceover

## Guidelines
- All audio must be in .mp3 format
- Sound effects should be short (under 5 seconds)
- Music should loop seamlessly
- All audio must be royalty-free or properly licensed
